# Android + Koin + Kotlin
Este repositório tem o objetivo de fornecer um projeto base com uso do Koin.

Outras libs e conceitos utilizados:
- AsyncTask;
- Room;
- Repository;
- Resource;
- Retrofit
- ViewModel;
- LiveData;
- RecyclerView.
